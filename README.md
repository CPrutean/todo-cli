# TODO-CLI

A random cli I felt like making to practice making my own query system

## How it works

You have a task that has some title, a description, a priority, and a some tags
Each time we create a task we create some folder and we can put things relating to that task in folder.

```
Task 1:
Title: Generate Task Readme
Description: Write the task-cli README.md file to describe how tasks work
Priority level: 0
Tags: Documentation, github, information
Additional resources: how_to_write_a_readme.txt, example.png
```

This will never get a GUI, there are enough TODO apps out there, I just wanna make a CLI to be cool and different. If you want to make a gui for this go ahead
